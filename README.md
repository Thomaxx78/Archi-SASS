# Archi-SASS

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Database

```bash
# Generate Prisma client
npx prisma generate

# Run migrations
npm run db:migrate

# Open Prisma Studio
npm run db:studio
```

## Code Quality

```bash
# Run linting
npm run lint

# Type checking
npm run typecheck

# Format code
npm run format:write
```

## Node mailer (MailDev)

```bash
docker run -p 1080:1080 -p 1025:1025 maildev/maildev
```
